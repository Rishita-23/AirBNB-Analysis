# 🏡 Airbnb Price Analysis

## 📖 Overview  
This project focuses on performing **Exploratory Data Analysis (EDA)** on Airbnb listing data to uncover **pricing patterns, demand trends, and key factors influencing listing prices**.

The goal is to derive **actionable insights** that help Airbnb hosts make better pricing and listing decisions.

---

## 🎯 Problem Statement  
Airbnb listing prices vary significantly based on multiple factors such as location, room type, and availability.  

The objective of this project is to:
- Understand **factors affecting Airbnb pricing**  
- Analyze **customer demand and availability trends**  
- Generate **data-driven insights for pricing optimization**

---

## 📊 Dataset  
The dataset contains detailed Airbnb listing information, including:

- Price  
- Neighbourhood  
- Room Type  
- Minimum Nights  
- Number of Reviews  
- Availability (365 days)  

---

## ⚙️ Technologies Used  
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Jupyter Notebook  

---

## 🔍 Exploratory Data Analysis (EDA)  

- Analyzed distribution of listing prices (highly skewed data)  
- Compared pricing across different room types  
- Studied neighbourhood-wise listing distribution  
- Examined availability trends to understand demand  

---

## 🛠 Data Preprocessing  

- Handled missing values  
- Removed duplicate entries  
- Converted data types for consistency  
- Applied **log transformation** to normalize skewed price distribution  

---

## 📊 Key Insights  

- Location plays a crucial role in determining listing prices  
- Entire homes/apartments are generally priced higher than private/shared rooms  
- Price distribution is right-skewed, indicating presence of high-value listings  
- High-demand areas tend to have lower availability  
- Listings with better engagement show stronger demand patterns  

---

## 🚀 Conclusion  

This project successfully:
- Identifies key drivers of Airbnb pricing  
- Highlights demand patterns across locations  
- Demonstrates how EDA can be used to extract meaningful insights from real-world datasets
